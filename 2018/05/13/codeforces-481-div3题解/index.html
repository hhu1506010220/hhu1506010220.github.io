codeforces-481-div3题解

2018-05-13

5.13 cf481 div3 6/7

比赛地址

A. Remove Duplicates

从左往右，抹除重复元素，留下最后一个不相同元素输出。

B. File Name

不能有连续超过三个X,输出需要移除个数.遇到连续的两个X就不断删后面的X即可。

C. Letters

给你room的总id，判断它在那个dormitory的第几个room.用前缀和记录每个宿舍所能容纳的结点的序号，用先lower_bound找到dormitory号，再减去sum[i-1]得到room号。

D. Almost Arithmetic Progression

给你某个序列，每个数只能动一次，并且只能+1、-1或不动。问你最少操作几次能使该序列变为等差序列。
因为是等差数列，所以只要前两项就确定了唯一的数列。只有9种情况。这样就可以枚举数列前两项再计算该数列的对应的操作次数，更新答案即可。

E. Bus Video System

有n个站点，汽车的容量是w ,现在给你每个站点，上车前和上车后人数的变化ai(1<=i<=n) ,让你求一开始车上有人的情况有多少种。如果没满足题意，输出0.
利用前缀和，求前缀和最大maxn，前缀和最小minn， 前缀和最大就是说，人数要<=w-maxn ,即上限.前缀和最小就是说，当变化ai为负时，说明人数要>=maxn，即下限.

F. Mentors

每个程序员都有一个skill值，当一个skill高的程序员把skill低的程序员当作导师的时候会吵架，反之则不会，还有就是给定的k对程序员是会吵架的，现在让你计算每一个程序员可以收多少徒弟.
模拟。创建一个struct node里面包含skill值和id（序号），然后对skill值进行排序，从前往后扫，计算出每一个id的可以收的徒弟数，然后在将输入的k对吵架的减去即可。
