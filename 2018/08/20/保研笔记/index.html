保研笔记

2018-08-20

本文是作者以及作者的同学，在准备保研过程中听了一些保研课而做的笔记，保存下来，分享给有需要的学弟学妹们。

认知

种类：

  1. 学习成绩
  2. 特长生（体育、文艺）
  3. 创新创业保研
  4. 免试推荐保留入学资格（先出去一两年）

天时：

  1. 从2014年开始，保研不分内外
  2. 保送研究生名额逐年递增
  3. 推荐不接受交叉（x

地利：
各学校基本情况及一些潜规则

  1. 看学校背景（双一流）
  2. 男生比女生更容易保研
  3. 清华电子系：唯一一个纯靠面试成绩（6轮）；千人计划的老师都有特招名额
  4. 可以不占用本校的推免名额：6所教育部直属的师范类院校
  5. 直博比学硕更加容易
  6. 保研夏令营

人和：
保研时应该具有的素质。一定要谦虚
硬实力：
学习成绩（必备项）
基础知识（必备项）
科研、学科竞赛（加分项）：看重你有没有动手实践
英语水平（加分项）
社会活动（加分项）
自身特长：一般不写
软实力：
材料准备
语言表达
逻辑思维

夏令营：
七八月份举办，申请时间五六月份
准备一份清单：

  1. 申请表
  2. 简历
  3. 个人陈述：1500字的，项目经历、科研规划，不是仅仅把简历展开
  4. 推荐信
  5. 本科三年成绩单
  6. 成绩排名证明
  7. 保研资格证明
  8. 英语成绩证明：不一定是四六级
  9. 奖学金证明及奖状复印件
  10. 学术论文：非常看重
    第一部分：学生充分了解导师
    第二部分：导师考察学生

一般三四月份联系导师，邮件正文、标题、回执

院校最好分成三份：
十拿九稳：
合我心意：
全力冲刺：

第一次填写和邮件不要去“全力冲刺”
工科一定要提前联系导师
如果官网找不到老师的邮箱，可以去找论文里的通讯作者

预推免：高校自己的系统

平行志愿不要随意填，还是要按照顺序

方法论

  1. 精准定位
    目标定位：学校、专业、导师
    问题定位：什么标准录取你、考核流程
  2. 清晰规划
    哪个方面不足？怎么弥补？
  3. 彻底执行
    做好了规划要立即开始
    持之以恒，坚持
  4. 事事有反馈
    失败以后要反思
    导师为什么不回邮件？
    面试为什么没有通过？
  5. 烂笔头
    一定要有记录

经历类问题，四大法则？

发邮件：
正文一定要大致分成三段

  1. 简单的自我介绍，说明意图
  2. 对自己特别突出的经历简单介绍
  3. 表达想去的意向和感谢
    最后附上个人简历以及证明材料
    不要点“已读回执”！

跨保：为什么要跨到这个专业？不能只是兴趣！成绩等

给同一个所的导师发邮件，一般第一个三五天没回才能发第二个

简历：这个经历有没有让我收获什么东西？能讲出什么？

论文：多看文章才能写出来。不一定要发表，能说得有理有据、头头是道就是好的科研经历

规划

  1. 二级学科
    学科门类 -> 一级学科 -> 二级学科
  2. 学硕和专硕
  • 学年制不同
  • 学费不同
  • 升至博士的途径不同
  1. 保研择校的正确思路
    先确定专业，top2可以例外
    专业 -> 城市 -> 学校 -> 导师
  2. 保研高校的地域影响
  3. 4+3保研规划法则
    四个问题
    1. 能去什么专业呢？能去到哪些个专业？如经济学下面有22个专业
    2. 要去什么专业呢？
    3. 你想要去什么城市？
    4. 你去的这个地方有哪些学校有你要去的专业呢？
      三个筛选条件
    5. 参照专业排名，后5个我不去了
    6. 招生人数有多少？
    7. 问学姐学长，看他们都去哪里了

导师

  1. 选导师时的两个衡量角度
  • 导师实验室的科研水平
  • 导师的个人特质

  1. 如何快速判断导师的学术能力？
    （详细剖析国内导师背景实力的层级关系）

    1. 院士（只有1000多人）：代表中国学术最高水平（资源不用愁）
    2. 杰青优青（自然基金委员会基金资助）：杰青每项400万，优青每项130万
    3. 长江学者（教育部评定，一般给社会科学）：特聘教授（聘任周期5年）、讲座教授（短期，不考虑）、青年学者
    4. 百人计划：中科院内部的研究水平
    5. 千人计划（中央）：海外引进的人才
    6. 万人计划（中央）：杰出人才（只有100人）、领军人才、青年拔尖人才
    • 863首席、专家：国家高技术研究发展计划

    • 973首席、专家：国家重点基础研究发展计划
      两者现已整合为国家重点研发计划，经费不用愁

      排序：

    1. 院士、万人计划中的杰出人才
    2. 国家重点研发计划首席、专家
    3. 长江特聘、千人计划、杰青、万人计划
    4. 更年轻的，优青、青千、万人计划中的青年拔尖、长江学者中的青年学者

  2. 导师的哪些个人特质需要你提前调研

    • 导师对学生的科研指导
    • 导师的性格（公平？易怒？）
    • 导师是否关注学生的未来成长（科研方向的选择、是否同意实习）
    • 导师是否有人情味（放假、生病、请假受批评）

简历

导师审核简历时间不会很长
简历的目的：突出你的个人中心，让导师快速的抓住录取你的点
导师最反感的四种简历类型

  1. 错误模板
  2. 毫无重点
  3. 内容空洞
  4. 疯狂自high

三大原则:

  1. 不造假原则
  2. 匹配性原则
    看看导师最近的项目，突出匹配的内容
    也可包括相关主干课的成绩

  3. 书面性原则

  4. 简历的内容和顺序

  5. 简历内容的三大原则
  6. 简历基本信息及注意事项
  7. 教育背景及注意事项
  8. 奖项荣誉及注意事项
  9. 科研能力及注意事项
    项目名称、工作内容、工作成果、起止时间、身份
  10. 综合能力&实践经历
  11. 附加项
    1. 研究生计划模板
      • 现状分析
      • 存在问题
      • 总体目标
    2. 自我评价
    3. 个人爱好

简历课

  1. 如何筛选/审核简历
    审核人：招生办老师or导师
    有些学校是把简历统一投送到招生办，而有些学校是直接投送到导师邮箱由导师审核
    审核时长很短-60s
    并不会因为你的简历页多字多就多花费时间，所以精炼和重点突出很关键
    基本信息10s
    科研经历40s
    其他经历10-20s
    简历目的：突出个人中心，让老师快速抓住录取你的点
  2. 导师反感的四种简历类型
    （1）错误模板
    封面
    不要封面（天马行空图片、座右铭、“个人简历”四个字……）
    多页
    不建议简历多于1页，不会因为多页而花费较多时间审核你的简历。
    图徽
    没必要放学校的校徽标志等。
    “创意”/奇葩
    黑色底板配红色字体！这种不可取，黑色底板对于PPT或简历都很难驾驭。禁止“微信对话”等类型的奇葩模板
    （2）毫无重点
    不是所有经历都要写在简历上，国家助学金不算奖，同一学年获得国家奖学金和国家励志时只写国奖就好
    （3）内容空洞
    不要写太多的自我评价，老师看来纯属自high，关键在数据和具体事例
    （4）视觉崩溃
    简历内容太多，看上去显得很优秀，但导师会觉得找不到重点所在。哪哪都是重点就是没有重点！
  3. 简历制作的原则和误区
  • 四大原则
    艺术品原则
    简单大方，把其当做一件艺术品，内外兼修，既要注重内容，又要注重外观
  • 不造假原则
    自己参与了但证书没名字，不能作假，这时可以把自己的任务工作量和收获写上。写在简历上的所有东西，一定要保证彻底明白的，不能为了简历显得丰满而都写上
  • 匹配性原则/重点性原则
    多次投递简历时，尽量不要用同一份简历，要针对性修改。让自己简历上的内容与导师研究方向或某个项目有一定的相关性，如自己会某个软件正好契合老师某个项目的需求，自己将来会很快上手。把与老师研究相关的，老师感兴趣的项目放在前面进行较为详细的介绍，把与老师研究方向相差较远的放到后面简要带过
  • 书面性原则
    书面化的语言，如“本人曾于大三学年在校学联任主席一职，负责XX活动的策划准备，取得XX成绩”
  • 两大误区
    老师最看重实力，所以只要有足够实力，简历是否精美对保研没有任何影响
    简历要尽可能详细，所有内容都要介绍非常清楚，这样老师才能更了解我
    老师没时间从论文一样的长篇大论中总结你的特点；言多必失；当导师通过简历已彻底了解你，就没有兴趣继续了解你了
  1. 优秀简历的具体内容
    按照下面的顺序先后写，不要颠倒，否则老师不容易找。 一般而言，中文简历即可，不需要英文。
    （1）基本项-基本信息
    包含内容：照片、姓名、性别、年龄、籍贯、政治面貌、邮箱、电话、成绩排名、英语成绩
    六级分高就写具体分数，不高就写六级二字，没过就不写英语成绩。六级分为几个等级“不及格，及格，480+，550+和600+”，具体可见历史推送
    注意事项：
    信息遗漏
    紧急联系人
    籍贯
    有些导师有老乡情怀
    读博意向
    如果有，就写上。导师更喜欢打算读博的学生
    证件照
    不能太随意，照相馆拍摄，画个淡妆，后期美修。
    邮箱超链接
    163邮箱，尽量不用QQ邮箱
    （2）重点项
    【教育背景】
    如果科研经历很丰富，可以把教育背景和基本信息放在一起。
    目的
    看学习成绩是否优秀
    看是否契合导师方向
    内容
    曾于XX学校XX专业学习，写明起止时间（即本科，不涉及中学）
    本科阶段的主要课程和学习成绩
    必要时：相关培训、高分的重要课程
    若想教育背景显得丰满，可以写曾经参加过的相关培训，如二学历、编程语言等。本科阶段非常重要而且自己考得不错的专业课写出来（此点要具体分析，不同研究方向的老师，看重的课程不一样）
    【科研能力】
    重点中的重点！！但也只需要展开其中一两项来写。不要炫技，只需要附上可以代表最高水平的作品集3-5份即可，放的东西越多机会反而越少！简历上的科研经历等，要在面试前重新熟悉一下
    每一项：项目名称，工作内容，工作成果，起止时间，你的身份/大标题下包含四个板块：项目内容，所需技能，个人贡献，项目成果
    未发表的论文，写明“哪一年，完成论文XXX题目，内容是……做了什么事”即可。大部分学生都没结项，尤其是大学生创新项目，一般一年。做的项目，如果时间长至1年等可以表明你的毅力的，就可以写明时间
    若空间限制只能一句话：摆数据
    如：曾带领5名团队成员，通过5个月的研发和测试，成功实现了一款可独立飞行的飞行器，在近5000个参赛队伍中跻身百分之五，获得一等奖
    【奖项荣誉】
    不要写优秀宿舍长、钢琴级别-附加项个人爱好
    时间倒序，大奖在前
    注意顺序，勿加冗余
    【专业技能】
    奖项荣誉和专业技能可以颠倒。切忌“精通XX”，要写“熟悉”。精通excel？那你会不会透视表？各种公式？
    英语能力
    计算机能力
    会计证等
    【实践经历】/综合能力/学生工作
    “列出每项活动，每项活动加1句话介绍”的做法不可取，因为老师不知道具体情况，不了解你在其中充当的角色和从中的收获​，更无法判断你实际上是否参加
    选择其中1-2个活动，具体介绍工作任务等
    老师注重：学生有无主观能动力-马不扬鞭自奋蹄，有时比科研经历更重要
    学生工作-摆数据
    2015-2016学年，担任学生会主席。（不好）要说明具体，如：曾任学生会副主席，在学院新年晚会中解决了所需的3万元经费问题
    捡重点写，不能头重脚轻。不能超过两条。科研经历占主要，这个为辅。
    （3）附加项
    一页纸没用完时（如实在没有科研经历），则可以写1-2点的附加项。
    【研究生规划】（首选）
    明确的规划：本人目前对编程很感兴趣，但缺乏一些想法或主动探索的科研能力，在未来三年中，我要在第一年做……第二年发表……文章，毕业后达到……的高度。
    现状分析
    自己的优势在哪，对哪方面更熟悉
    存在问题
    在哪方面补充加强
    总体目标
    研究生三年的总体目标，如何一步步走？
    【自我评价】
    清水老师的经验：大部分简历的自我评价，导师们都不看。因为千篇一律。建议：每夸赞一句就跟一个实例，如本人毅力较强坚持不懈，曾学习钢琴持续5年。
    【个人爱好】
    套路：导师也乐意把自己的兴趣爱好写在网站上，如果和导师有相同的爱好，如都喜欢NBA，就可以写，一定可以就这个话题聊下去
  2. 制作简历的步骤
    每次的简历版本都要保存起来Vx
    （1）成立自己的简历库
    财政、会计、金融三个领域文件夹，每个文件夹都有对应的学校人大、上财、央财文件夹，其内又可以有多个Vx，在多次投递简历时不断完善美化简历，就是一个增强自信的成熟过程
    （2）绘制初稿书写脑图
    回顾大学做过的所有事情（不论大小），按照时间顺序完整写一遍
    按照要写的内容（基本项、重点项、附加项）进行分类，梳理出属于自己的逻辑思维脑图（推荐：网页版百度脑图）
    （3）分析与匹配需求
    优先匹配相关性经历
    如：金融专硕的实习经历占主要部分，经济学学硕的科研经济占主要部分
    挖掘与导师相关研究方向的科研经历
    如：与导师相关的且分数很高的学科成绩要重点突出
    挖掘与导师相关的兴趣爱好
    （4）美化润色
    修内
    做好后，多问自己为什么，猜测老师会问什么问题。准备英文的提问和英文的回答。
    修外
    版面风格
    表格式简历早已大跌眼镜！而且有非常多的条框约束，会浪费很多空间
    黑白系
    色彩系
    淘宝的简历模板多为彩色系
    让人心情愉悦，推荐
    创意系
    简历网站推荐：千图网、向日葵简历、五百丁、淘宝
    比较好的模板，需要花几块钱购买
    （5）投递/打印简历
    版式
    PDF
    发送老师时也务必PDF，最好选择没有之一
    彩打
    彩打的简历比黑白打印的简历更能突显诚意，更容易得到青睐！
    好纸
    用的纸不能太随意，推荐信也如此。能用80g的纸就不要70g，能用含棉25%就不用一般纸
    其他
    文件名：申请攻读博士/硕士研究生-学校-姓名
    单独附件：简历和其他文档分开，在同一邮件中以不同附件发送
    先练手：第一次投递简历，不要直接投给心仪的导师，先拿其他老师练练手
  3. 个人总结：
    政治面貌：写“中共党员”而非“党员”
    版式上，各个大标题和分区线在同一水平，而非上下排列，可以更节省空间
  4. 其他杂碎信息
    清华更看重裸绩排名，北大更看重综合排名
    跨专业读研，不要强调兴趣，没用！相关能力才重要
    本校的学长学姐，以点带面，了解其他师兄姐
    单纯的参观某公司，没必要写在简历上，要真正实习才行
