Ubuntu下安装MediaWiki

2018-08-26

最近老板给了一个任务，让我利用wiki搭一个实验室的主页网站。先后实验了XWiki、ScrewTurnWiki、JspWiki、DokuWiki、MediaWiki,最后选择了MediaWiki。

需求

  • 样式需求：参照：http://lamda.nju.edu.cn/。
  • 性能需求：占用尽可能小的性能。
  • 备份需求：由于实验室机器经常坏..所以需要容易备份文件。
  • 功能需求：关于用户设置，一般的wiki都是所有人可以编辑的（包括外人），要求可以设置比如仅实验室的学生有操作权限。

Media对应的特点

  • 样式有现成的皮肤与lamda主页相像，可以进行css修改。
  • 性能不大，apache+php+mysql驱动，我感觉实验室主页这样的访问量不会是爆发式的那种，所以应该是可以满足性能需求的。
  • 数据好备份，他的核心应该就是一个localsetting.php文件。文件直接打包即可。
  • 有用户组的，可以根据用户组设置权限。

所以就愉快地决定用Media啦！

安装过程

安装apache2、php7.0、mysql

Web服务器:apache2 server(sudo apt-get install apache2)
PHP语言:php (sudo apt-get install php7.0 php7.0-mysql php7.0-xml php7.0-mbstring)
数据库:mysql(sudo apt-get install mysql-client mysql-server)
当然上述的除了php没的选，web服务器(apahce,iis,nginx,lightttpd)和数据库(msysqk,PostgreSQL,Sqlite3,Oracle,ms sql server)都是具有多个选择的。

配置DB

登录MySQL数据库：mysql -u root -p，为MediaWiki创建专用DB及用户，
mysql> CREATE DATABASE wiki;
mysql> CREATE USER wikiuser@localhost IDENTIFIED BY ‘wikipass’;
mysql> GRANT index, create, select, insert, update, delete, alter, lock tables on wiki.* TO wikiuser@localhost;
mysql> FLUSH PRIVILEGES;
mysql> exit

下载Mediawiki包

下载mediawiki的tar包(http://www.mediawiki.org/wiki/Download)
wget http://releases.wikimedia.org/mediawiki/1.31/mediawiki-1.31.0.tar.gz
tar -xvzf mediawiki-.tar.gz
sudo mkdir /var/lib/mediawiki
sudo mv mediawiki- /var/lib/mediawiki
并利用ln命令将其关联同步到apache2的服务器主目录下（/var/www/html）这里说明下Ubuntu16.04是有html的，以往版本可能没有html文件夹
cd /var/www/html
sudo ln -s /var/lib/mediawiki mediawiki
在浏览器中输入http://[domain]/mediawiki/,domain可以是域名或直接的ip地址，比如 http://222.42.14.9/mediawiki/

开始安装Media,进行配置

访问 http://localhost/mediawiki，即可开始配置。
