Java爬虫将CSDN博客自动转化为hexo所需md文件

2018-03-19

博主从大一到现在的所有博客全部都在CSDN上，由于打算弃用CSDN专用Hexo，所以要对CSDN做一个文章导出功能并转为markdown格式。官方提供了工具，然而试了一下并不可行。有网友自己提供了工具，是用python写的，但是在尝试过之后发现网上的工具包都是基于python2.7版本，依赖于beautifulsoup包，并且对CSDN主题仅限于”碧海蓝”主题，尝试多次之后并不能爬到东西。所以参照网友的方法，改用java实现这个功能。

大题思路如下:




首先使用Jsoup去获取页面并解析，然后将解析出的html代码转换成MD文件存储到本地(默认设置D:\data\csdn_blogs)，最后移动到hexo文件夹下的对应位置，执行hexo g和hexo d命令发布，网站更新。

在具体实现的过程中，逻辑主要就是先确定我博客的主界面(文章列表)，通过解析”共xx页”来解析总页数，依次获得每个列表页的地址存入set中。结果在处理这个的过程中发现CSDN改版了。。原来的分页”共x页”给改没了，于是就手动设置了一下总页数。。然后对于每页，就通过解析元素可以获得每个具体文章页的网址。至此就得到了全部文章页的网址。

之后进行到转码的时候 采用了一个开源的jHTML2Md 拿来现套 后续学习一下 https://github.com/pnikosis/jHTML2Md

最终我们可以看到 可以成功的转换为md文件


但是需要注意的是，在设定结构化时，我们对date的处理还不是很对，只能转换成xxxx年xx月xx日的形式，但是hexo要求必须是xxxx-xx-xx的格式，于是我就手动改了一下。。

然后我们将这些md文件移动到/source/_post文件夹下，只需要hexo g一下s一下…大功告成。
以上过程略显粗糙，如有不妥之处欢迎批评指正。
