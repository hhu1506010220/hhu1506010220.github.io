HDOJ 3791 二叉搜索树（二叉树）

2016-07-31

二叉搜索树

Time Limit: 2000/1000 MS (Java/Others) Memory Limit: 32768/32768 K (Java/Others)
Total Submission(s): 4561 Accepted Submission(s): 2008
Problem Description

判断两序列是否为同一二叉搜索树序列

Input

开始一个数n，(1<=n<=20) 表示有n个需要判断，n= 0 的时候输入结束。
接下去一行是一个序列，序列长度小于10，包含(0~9)的数字，没有重复数字，根据这个序列可以构造出一颗二叉搜索树。
接下去的n行有n个序列，每个序列格式跟第一个序列一样，请判断这两个序列是否能组成同一颗二叉搜索树。

Output

如果序列相同则输出YES，否则输出NO

浙大计算机研究生复试上机考试-2010年
思路：

这题的题意就是，给出一串数字，可以建立一颗二叉排序树，后边又给出n串数字， 判断这n串数字建立起来的二叉排序树是否和原来的一致。 首先用给出的数字串建立一颗二叉排序树，遍历，得到遍历结果，以此为标准; 然后对后边给出的每一串数字，建立二叉排序树，按照和上面同样的方法进行遍历， 如果遍历的结果和标准结果一样，说明建立的二叉排序树和标准的是一样的，输出YES， 如果不一样，说明建立的和标准的不一样，输出NO。本来一开始没学数据结构的时候的想法是，将输入的数组进行排序，看数组从小到大之后顺序是否一样就能确定树一不一样。但是后来知道二叉搜索树的创建之后之后才恍然大悟。二叉搜索树要求每个结点左边的数都比右边小，就比如567432和567423，他们是有区别的。所以还是按照常规来做。
